# Our first program

<p style={{ width: '100%' }}>Place this code into the text editor window of the Arduino IDE.</p>

```
int led = 13;

void setup() {pinMode(led, OUTPUT)}
void loop() {
  digitalWrite(led, HIGH);
  delay(1000);
  digitalWrite(led, LOW);
  delay(1000);
}
```
